# 热工计算说明

## 热量衡算

热侧煤油放热量按下式计算：

`Q = m_h * c_p,h * (T_h,in - T_h,out)`

其中：

- `Q` 为热负荷，单位 `W`
- `m_h` 为煤油质量流量，单位 `kg/s`
- `c_p,h` 为煤油平均温度下的定压比热，单位 `J/(kg·K)`

冷却水流量按下式计算：

`m_c = Q / [c_p,c * (T_c,out - T_c,in)]`

## LMTD

采用逆流对数平均温差：

`ΔT1 = T_h,in - T_c,out`

`ΔT2 = T_h,out - T_c,in`

`LMTD = (ΔT1 - ΔT2) / ln(ΔT1 / ΔT2)`

当 `ΔT1` 与 `ΔT2` 很接近时，采用极限形式，直接取二者平均值。

## 修正系数 F

本程序默认采用 1 壳程、2 管程课程设计结构，先定义：

`R = (T_h,in - T_h,out) / (T_c,out - T_c,in)`

`P = (T_c,out - T_c,in) / (T_h,in - T_c,in)`

然后采用教材常见的 `R-P` 解析式计算 `F`。有效温差为：

`ΔT_eff = F * LMTD`

## 总传热系数与面积

程序采用两阶段策略：

1. `U_assumed`：从课程经验值读取，用于面积初选。
2. `U_calculated`：用冷热侧膜系数、污垢热阻和壁热阻进行校核。

外表面积基准下的总热阻写为：

`1 / U_o = 1 / h_o + R_f,o + (d_o ln(d_o / d_i)) / (2 k_wall) + (d_o / d_i) * (1 / h_i + R_f,i)`

最终所需面积为：

`A = Q / (U * ΔT_eff)`

## 变量释义

- `h_i`：管程水侧膜传热系数
- `h_o`：壳程煤油侧膜传热系数
- `R_f,i`：管程污垢热阻
- `R_f,o`：壳程污垢热阻
- `k_wall`：管壁导热系数
