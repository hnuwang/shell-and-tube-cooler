# 封面后摘要 + 设计计算结果总表 + 结论

## 摘要

本课程设计以固定管板式管壳式煤油冷却器为研究对象，完成了传热计算、结构计算和阻力计算的程序化实现。设计任务是将质量流量为 14 t/h 的煤油由 150 ℃ 冷却至 50 ℃，冷却介质为水，进出口温度分别为 20 ℃ 和 30 ℃；煤油工作表压力为 0.1 MPa，水工作表压力为 0.3 MPa。程序采用 Python 语言编写，物性参数依据题目给出的煤油和水物性表整理后，通过线性插值获得；计算过程中统一采用 SI 单位，并将关键中间量以结构化结果输出，便于复核和整理说明书。

本设计按课程设计常规步骤进行。首先根据煤油平均温度求取煤油物性，并据此完成热负荷计算；然后根据冷却水平均温度求取水物性，计算所需冷却水流量；在此基础上计算逆流对数平均温差及修正系数，采用经验总传热系数进行面积初选。随后对候选管长和管数进行离散筛选，计算实际传热面积、壳径、挡板间距及相关几何参数，并结合冷热侧膜传热系数、污垢热阻和壁热阻对总传热系数进行校核。最后完成管程和壳程压降计算，并与许用值进行比较，判断所选方案是否满足设计要求。

计算结果表明，在默认课程设计假设下，最终选定方案为：管长 3.0 m，总管数 178 根，每程 89 根，实际传热面积 31.87 m²，面积裕量约 5.4%。热负荷约为 927.79 kW，所需冷却水流量约为 22.19 kg/s；逆流对数平均温差为 64.92 K，修正系数为 0.9547，有效传热温差为 61.98 K；校核总传热系数约为 512.94 W/(m²·K)。在阻力方面，管程流速约为 1.24 m/s，壳程流速约为 0.43 m/s，管程压降约为 10.23 kPa，壳程压降约为 2.38 kPa，均低于设定许用值。结果说明该方案在面积、流速及压降等方面均满足课程设计要求，可作为本题推荐设计方案。

关键词：固定管板式换热器；煤油冷却器；传热计算；结构计算；阻力计算；Python

## 结论

本课程设计完成了固定管板式管壳式煤油冷却器的热工、结构及阻力计算，并利用 Python 程序实现了物性插值、面积初选、结构筛选、总传热系数校核和压降校核等完整流程。计算结果表明，在题目给定工况及课程设计假设条件下，所选换热器方案能够满足煤油由 150 ℃ 冷却至 50 ℃ 的要求。

最终确定的结构方案为：采用 1 壳程、2 管程，冷却水走管程、煤油走壳程；换热管外径 19 mm、内径 16 mm、管长 3.0 m，总管数 178 根。该方案的实际传热面积为 31.87 m²，大于校核所需面积 29.18 m²，面积裕量约为 5.4%，说明换热面积满足要求且余量适中。与此同时，管程流速与壳程流速均位于设定允许范围内，管程压降 10.23 kPa、壳程压降 2.38 kPa，均低于许用压降，因此阻力校核合格。

综合分析可知，本设计所得方案在传热能力、结构尺寸和流动阻力等方面均满足课程设计要求，所采用的计算方法和程序流程具有较好的条理性、可复核性和实用性。由于本设计采用的是课程设计版简化模型，壳程传热与阻力计算、管束直径估算及部分经验参数仍具有一定近似性，但这些简化不影响课程设计层面的方案判断。总体而言，本程序能够较好地完成本题设计任务，所得结果可作为课程设计说明书和程序提交的主要依据。
