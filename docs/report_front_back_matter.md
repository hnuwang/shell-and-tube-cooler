# 前言 + 设计任务书 + 参考文献素材

## 前言

换热器是化工、石油、轻工、能源及制冷等工业领域中广泛应用的重要热工设备，其主要作用是实现不同流体之间的热量交换，以满足加热、冷却、冷凝或蒸发等工艺要求。管壳式换热器因其结构成熟、适用范围广、承压性能较好、制造与维护经验丰富，在工程实际中具有十分重要的地位。其中，固定管板式换热器结构简单、造价较低、制造方便，是课程设计中最常见也最具有代表性的换热设备之一。

本次课程设计以固定管板式管壳式煤油冷却器为对象，要求在给定工况下完成热工计算、结构计算和阻力计算，并通过编程方式实现整个设计过程。与单纯手工计算相比，程序化设计不仅能够提高计算效率，还能够使中间量更清晰、结果更易复核，并为后续方案比较、参数修改和工程分析提供便利。

本设计采用 Python 语言进行实现，计算过程中统一使用 SI 单位，物性参数依据题目给出的煤油和水物性数据表进行整理，并通过线性插值获得代表物性。程序按照课程设计的常规步骤进行组织，包括热负荷计算、冷却水流量计算、对数平均温差及修正系数计算、经验总传热系数初选、几何结构筛选、总传热系数校核以及管程和壳程压降计算。通过这一过程，可以系统地训练对换热器课程知识的综合运用能力，同时加深对工程假设、模型简化及程序实现逻辑的理解。

由于课程设计的目标是建立清晰、完整、可解释的设计流程，因此本程序优先采用教材中常见、便于说明和复核的简化关联式，而未引入更为复杂的工业详细设计模型。虽然这种方法在工程精度上具有一定近似性，但对于课程设计层面的方案判断与计算训练是适用且合理的。希望通过本次设计，能够进一步提高对管壳式换热器设计方法、传热过程分析以及工程程序化实现的认识。

## 设计任务书

### 一、设计题目

固定管板式管壳式煤油冷却器设计程序

### 二、设计目的

1. 通过课程设计掌握管壳式换热器的基本设计步骤和计算方法。
2. 熟悉固定管板式换热器在传热计算、结构计算和阻力计算中的主要参数与公式。
3. 学会将工程计算过程程序化，实现物性整理、参数计算、方案筛选和结果输出。
4. 提高综合运用传热学、化工原理及工程制图相关知识分析和解决实际问题的能力。
5. 培养工程化表达能力，为编写课程设计说明书打下基础。

### 三、设计内容

根据题目给定工况，设计一台固定管板式管壳式煤油冷却器，完成以下内容：

1. 传热计算
   根据煤油和冷却水工况，完成热负荷计算、冷却水流量计算、对数平均温差计算、修正系数计算和传热面积计算。
2. 结构计算
   根据换热面积要求，确定换热管规格、管长、管数、壳体尺寸及挡板间距等主要结构参数。
3. 阻力计算
   分别计算管程压降和壳程压降，并校核其是否满足许用要求。
4. 程序实现
   采用 Python 语言编写计算程序，实现物性插值、热工计算、结构筛选、阻力校核和结果表输出。
5. 说明书撰写
   编写课程设计说明书，内容包括设计依据、物性数据整理、计算过程、结果分析和设计结论。

### 四、原始数据

- 煤油质量流量：14 t/h
- 煤油入口温度：150 ℃
- 煤油出口温度：50 ℃
- 冷却水入口温度：20 ℃
- 冷却水出口温度：30 ℃
- 煤油工作表压力：0.1 MPa
- 水工作表压力：0.3 MPa

### 五、设计要求

1. 按照课程设计步骤独立完成固定管板式管壳式煤油冷却器的设计计算。
2. 使用 Python 编写设计程序，不得直接硬编码最终结果。
3. 物性参数应根据题目提供的煤油、水物性表整理后，通过插值获得。
4. 程序应能输出传热计算、结构计算和阻力计算所需的关键结果与中间量。
5. 若候选方案不满足面积、流速或压降约束，应自动筛选下一方案或给出失败原因。
6. 最终提交内容包括程序代码和课程设计说明书。

### 六、设计成果要求

本次课程设计最终应形成以下成果：

1. 一套可运行的 Python 程序；
2. 一份完整的课程设计说明书；
3. 说明书中应包含输入参数表、物性参数表、热工结果表、结构结果表和阻力校核表；
4. 程序和说明书中的数据、单位和计算逻辑应保持一致。
