# 固定管板式管壳式煤油冷却器课程设计说明书正文初稿

## 摘要

本设计题目为固定管板式管壳式煤油冷却器设计。设计任务是将质量流量为 14 t/h 的煤油由 150 ℃ 冷却至 50 ℃，冷却介质为冷却水，进出口温度分别为 20 ℃ 和 30 ℃。根据题目要求，程序需完成传热计算、结构计算和阻力计算，并输出可用于课程设计说明书整理的中间量和结果表。

本程序采用 Python 3.11 实现，统一使用 SI 单位，物性参数根据题目给出的煤油和水物性表录入 CSV 后进行线性插值。计算流程按照课程设计常规步骤展开，即先进行热负荷与平均温差计算，再采用经验总传热系数进行面积初选，随后对候选管长与管数进行结构筛选，最后结合膜传热系数、污垢热阻和壁热阻对总传热系数进行校核，并完成管程与壳程压降计算。

在本次默认设计工况下，程序筛选出的可行方案为：管长 3.0 m，总管数 178 根，每程 89 根，实际传热面积约 31.87 m²，面积裕量约 5.4%；校核总传热系数约为 512.94 W/(m²·K)；管程流速约 1.24 m/s，壳程流速约 0.43 m/s；管程压降约 10.23 kPa，壳程压降约 2.38 kPa。结果表明，该方案满足课程设计中面积、流速及压降等约束要求。

## 1. 设计任务与原始数据

### 1.1 设计任务

设计一台固定管板式管壳式煤油冷却器，完成如下三部分计算：

1. 传热计算
2. 结构计算
3. 阻力计算

### 1.2 已知工况

- 煤油质量流量：14 t/h，即 3.889 kg/s
- 煤油入口温度：150 ℃
- 煤油出口温度：50 ℃
- 冷却水入口温度：20 ℃
- 冷却水出口温度：30 ℃
- 煤油工作表压力：0.1 MPa
- 水工作表压力：0.3 MPa

### 1.3 设计要求

- 按课程设计步骤完成程序化计算
- 物性参数由题目表格整理后插值获得
- 输出结果应便于复核和撰写说明书
- 采用课程设计版简化关联式，不追求工业级精细模型

## 2. 设计方案与基本假设

### 2.1 工艺方案选择

本设计采用固定管板式管壳式换热器，取冷却水走管程、煤油走壳程。这样布置的主要原因是：冷却水黏度较低、给热系数较高，适合在管内流动以提高传热效果；同时水侧更易清洗，便于课程设计中对管程流速和压降进行校核。煤油布置在壳程，有利于结合简化的壳程横掠模型完成计算。

### 2.2 结构与计算假设

为保证程序结构清晰、便于解释，本设计采用以下工程假设：

- 默认采用 1 壳程、2 管程
- 管外径取 19 mm，管内径取 16 mm
- 管长候选取 1.5 m、2.0 m、3.0 m、4.5 m、6.0 m
- 管间距按 `pitch_ratio = 1.25` 确定
- 管束排列角取 30°，按三角形排列近似处理
- 挡板间距按壳径的 0.4 倍估算，并限制在给定范围内
- 管壁导热系数及污垢热阻取课程设计经验值
- 壳程横流面积和壳程当量直径统一由几何模块给出，传热与阻力共用

上述假设并非题目原始数据，而是课程设计中为形成可计算方案而引入的工程近似。这些参数在程序中统一集中于 `config.py`，便于后续修改和复核。

## 3. 物性参数整理与取值方法

### 3.1 物性数据来源

煤油和水的物性参数均来自题目 PDF 中给出的表格。表中包含温度、密度、定压比热、动力黏度、运动黏度、导热系数、热扩散率和 Pr 数等数据。

### 3.2 单位统一

为保证计算一致性，程序对题目中的数据进行了统一换算：

- 温度统一以 K 存储
- 比热由 kJ/(kg·K) 换算为 J/(kg·K)
- 黏度按 Pa·s 录入
- 导热系数按 W/(m·K) 录入
- 运动黏度按 m²/s 录入
- 热扩散率按 m²/s 录入

### 3.3 插值方法

程序按流体平均温度在线性插值获得物性参数。对本题，热侧代表温度取：

`T_h,avg = (150 + 50) / 2 = 100 ℃`

冷侧代表温度取：

`T_c,avg = (20 + 30) / 2 = 25 ℃`

插值得到的代表物性如下：

- 煤油密度约 765.47 kg/m³
- 煤油动力黏度约 5.585 × 10⁻⁴ Pa·s
- 煤油定压比热约 2385.75 J/(kg·K)
- 煤油导热系数约 0.1041 W/(m·K)
- 煤油 Pr 数约 12.71

- 水密度约 996.93 kg/m³
- 水动力黏度约 9.081 × 10⁻⁴ Pa·s
- 水定压比热约 4180.85 J/(kg·K)
- 水导热系数约 0.6072 W/(m·K)
- 水 Pr 数约 6.28

## 4. 热工计算

### 4.1 热负荷计算

热侧煤油放热量按下式计算：

`Q = m_h * c_p,h * (T_h,in - T_h,out)`

代入数据：

- `m_h = 3.889 kg/s`
- `c_p,h = 2385.75 J/(kg·K)`
- `T_h,in - T_h,out = 100 K`

得：

`Q = 927791.67 W`

即热负荷约为：

`Q ≈ 927.79 kW`

### 4.2 冷却水流量计算

冷却水所需质量流量按下式计算：

`m_c = Q / [c_p,c * (T_c,out - T_c,in)]`

代入：

- `Q = 927791.67 W`
- `c_p,c = 4180.85 J/(kg·K)`
- `T_c,out - T_c,in = 10 K`

得：

`m_c = 22.19 kg/s`

说明在给定水温升条件下，需要约 22.19 kg/s 的冷却水流量才能带走热侧释放的热量。

### 4.3 对数平均温差与修正系数

按逆流计算：

`ΔT1 = T_h,in - T_c,out = 150 - 30 = 120 K`

`ΔT2 = T_h,out - T_c,in = 50 - 20 = 30 K`

则：

`LMTD = (ΔT1 - ΔT2) / ln(ΔT1 / ΔT2) = 64.92 K`

由于换热器为 1 壳程、2 管程，需引入修正系数 `F`。程序按课程设计常用 `R-P` 解析式计算，得到：

`F = 0.9547`

因此有效平均温差为：

`ΔT_eff = F * LMTD = 61.98 K`

### 4.4 经验总传热系数初选

课程设计初选时，采用经验总传热系数：

`U_assumed = 300 W/(m²·K)`

由此得到初算所需面积：

`A_initial = Q / (U_assumed * ΔT_eff)`

可得初选面积约 49.90 m²。该步骤的目的在于给出结构筛选的初始量级，而不是作为最终结果。

### 4.5 总传热系数校核

在几何方案确定后，程序进一步根据膜传热系数、污垢热阻和壁热阻进行分项校核。水侧膜传热系数采用管内流动分段关联式，煤油侧膜传热系数采用课程设计版壳程简化关联式。计算得到：

- 管程水侧膜传热系数约为程序内计算值
- 壳程煤油侧膜传热系数约为程序内计算值
- 校核总传热系数：

`U_calculated = 512.94 W/(m²·K)`

再按校核后的总传热系数计算所需面积：

`A_required = Q / (U_calculated * ΔT_eff) = 29.18 m²`

由此可见，经验总传热系数偏保守，而校核后的总传热系数较高，使得最终所需面积显著下降。这符合课程设计中“先保守初选，再按分项热阻校核”的思路。

## 5. 结构计算与方案筛选

### 5.1 候选生成原则

程序以所需面积为基础，对多个管长候选进行离散筛选。对每个管长，先按外表面积公式反推理论管数，再修正为可被管程数整除的总管数，随后计算实际面积、流速及壳程几何量，并检查如下约束：

- 面积裕量是否位于设定区间
- 管程流速是否在允许范围内
- 壳程流速是否在允许范围内

若候选不满足要求，则增加一组完整管程的管数继续迭代，直到找到合格方案或所有候选失败。

### 5.2 最终筛选结果

程序最终筛选出的方案如下：

- 管外径：19 mm
- 管内径：16 mm
- 管长：3.0 m
- 总管数：178 根
- 每程管数：89 根
- 实际传热面积：31.87 m²
- 面积裕量：5.4%

该结果说明实际面积略大于理论所需面积，既满足换热要求，又没有出现明显过度保守的面积超配。

### 5.3 壳程几何结果

根据统一几何定义，程序得到：

- 壳体内径：0.3847 m
- 挡板间距：0.1539 m

这些几何量既用于热工校核，也用于后续壳程阻力计算，从而保证了全流程计算口径一致。

## 6. 阻力计算与校核

### 6.1 管程流速与阻力

由冷却水流量和每程管数计算得管程流速：

`v_tube = 1.244 m/s`

据此可得管程雷诺数：

`Re_tube = 21849`

可判断管内水流处于湍流区，适合采用湍流摩擦因子关联式。程序最终计算得到管程压降：

`ΔP_tube = 10227 Pa = 10.23 kPa`

该值远低于默认许用压降 50 kPa，说明管程阻力满足要求。

### 6.2 壳程流速与阻力

由煤油流量和壳程横流面积计算得壳程速度：

`v_shell = 0.429 m/s`

进一步得到壳程雷诺数：

`Re_shell = 7933`

程序采用课程设计版简化壳程压降模型进行计算，得：

`ΔP_shell = 2382.88 Pa = 2.38 kPa`

同样低于默认许用压降 40 kPa，因此壳程阻力也满足设计要求。

因此，可以认为本次筛选出的结构方案满足题目要求，能够作为课程设计推荐方案。

### 7.3 简化模型影响分析

需要说明的是，本程序采用的是课程设计版简化模型，其特点是公式简单、物理意义清楚，便于解释和复核，但与工业详细设计相比仍存在一定近似性，主要体现在以下方面：

- 壳程传热和阻力采用简化 Kern 思路，而未使用更复杂的 Bell-Delaware 模型
- 管束直径和壳径采用课程设计经验近似，而非标准排管表精确查取
- 污垢热阻、管壁导热系数等参数采用经验值
- 物性参数按平均温度插值，未考虑沿程连续变化

这些简化不会影响课程设计的教学目的，但在说明书中应明确其适用范围和工程含义。

## 8. 程序结构与测试说明

### 8.1 程序结构

程序主要由以下模块组成：

- `properties.py`：物性表读取与插值
- `thermal_design.py`：热负荷、LMTD、修正系数和总传热系数计算

- `main.py`：主流程集成
